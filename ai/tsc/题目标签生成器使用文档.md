# 题目标签生成器使用文档

## 功能概述

基于 AI 的题目智能标签生成系统，支持：
- 自动识别学科、知识点、难度、题型、能力素养
- 支持图片题目（视觉模型识别）
- 增量合并标签（保留原有标签）
- 复用已有标签库（避免同义异构）
- 批量多线程处理


## Python API 使用

### 1. 处理单个题目

```python
from ai.tag_processor import process_tags_for_question
from ai.base import AI, ReasoningEffort

# 配置 AI
ai = AI(
    model="doubao-seed-2-0-pro-260215",
    reasoning_effort=ReasoningEffort.medium,
    max_output_tokens=4096,
)

# 处理单个题目
result = process_tags_for_question(
    data_dir="./data",                    # 数据目录
    question_id="20260314100325",         # 题目ID
    prompt_path="./ai/tsc/题目标签.txt",  # 提示词文件
    ai=ai,
    replace=False,                        # False=增量合并, True=替换
    skip_if_has_valid_tags=False,         # 是否跳过已有规范标签的题目
)

if result:
    print(f"生成 {len(result.tags)} 个标签:")
    for tag in result.tags:
        print(f"  - {tag}")
```

### 2. 批量处理

```python
from ai.tag_processor import batch_process_tags
from ai.base import AI, ReasoningEffort

ai = AI(
    model="doubao-seed-2-0-pro-260215",
    reasoning_effort=ReasoningEffort.medium,
)

# 批量处理所有题目
results = batch_process_tags(
    data_dir="./data",
    prompt_path="./ai/tsc/题目标签.txt",
    ai=ai,
    replace=False,                        # 增量合并
    skip_if_has_valid_tags=True,          # 跳过已有规范标签
    max_workers=3,                        # 并发数
)

print(f"成功: {len(results['success'])} 个")
print(f"失败: {len(results['failed'])} 个")
print(f"跳过: {len(results['skipped'])} 个")
```

### 3. 批量处理指定题目

```python
from ai.tag_processor import batch_process_tags

# 只处理指定的题目ID
results = batch_process_tags(
    data_dir="./data",
    prompt_path="./ai/tsc/题目标签.txt",
    question_ids=["20260314100325", "20260314100415"],
    max_workers=3,
)
```

### 4. 收集已有标签库

```python
from ai.tag_processor import collect_all_tags

# 获取所有题目的标签
all_tags = collect_all_tags("./data")
print(f"标签库共 {len(all_tags)} 个标签")
```

## 参数说明

### AI 配置参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `model` | str | `doubao-seed-2-0-pro-260215` | AI 模型 |
| `reasoning_effort` | str | `medium` | 推理深度: `minimal`, `low`, `medium`, `high` |
| `max_output_tokens` | int | `4096` | 最大输出 Token 数 |

### process_tags_for_question 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `data_dir` | str | 必填 | 数据目录路径 |
| `question_id` | str | 必填 | 题目 ID |
| `prompt_path` | str | 必填 | 提示词文件路径 |
| `ai` | AI | None | AI 配置对象 |
| `replace` | bool | False | True=替换标签, False=增量合并 |
| `skip_if_has_valid_tags` | bool | False | 是否跳过已有规范标签的题目 |

### batch_process_tags 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `question_ids` | list | None | 指定题目ID列表，None=处理所有 |
| `max_workers` | int | 3 | 并发线程数 |

## 标签格式规范

生成的标签遵循四级分类体系：

### 1. 知识点标签
```
知识点::<学科>::<模块>::<章节>::<具体知识点>
```
示例：
- `知识点::数学::解三角形::余弦定理`
- `知识点::生物::稳态与环境::生态系统::能量流动`
- `知识点::化学::有机化学::羟基::取代反应`

### 2. 难度标签
```
难度::<难度等级>::<特征描述>
```
难度等级固定为：`容易`、`基础`、`中档`、`较难`、`困难`

示例：
- `难度::基础::概念辨析类`
- `难度::中档::多考点综合`
- `难度::困难::高考压轴类`

### 3. 题型标签
```
题型::<一级题型>::<二级题型>
```
示例：
- `题型::选择题::单项选择题`
- `题型::解答题::高考真题`
- `题型::实验题::探究性实验`

### 4. 能力素养标签
```
能力素养::<学科>::<核心素养>::<具体能力>
```
示例：
- `能力素养::数学::逻辑推理`
- `能力素养::生物::科学思维::图表分析`
- `能力素养::语文::思维发展与提升::归纳概括`

## 常见问题

### Q: 如何重新生成某个题目的标签？

```bash
# 直接运行，会增量合并
python test_tag_processor.py <题目ID>
```

### Q: 如何只处理没有规范标签的题目？

```python
# 使用 skip_if_has_valid_tags=True
batch_process_tags(
    ...,
    skip_if_has_valid_tags=True,  # 自动跳过已有规范标签的题目
)
```

### Q: 题目只有图片没有文字怎么办？

系统会自动提取图片并传给 AI 视觉模型进行识别，无需额外配置。

### Q: 如何修改提示词？

编辑 `ai/tsc/题目标签.txt` 文件即可。

### Q: 标签库在哪里？

标签库是从所有 `data/*.json` 文件的 `tags` 字段动态收集的，无需手动维护。
